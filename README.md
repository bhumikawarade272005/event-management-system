# Event Management System

This is a web-based Event Management System project.

## Technology Used
- Python (Flask)
- HTML
- CSS
- JavaScript

## Project Structure
- app.py : Backend logic
- templates/ : HTML files
- static/images/ : Images

## How to Run Project
1. Install Python
2. Run app.py
3. Open browser and go to http://127.0.0.1:5000
